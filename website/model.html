Modelling the uncertainty and stochasticity

We wish to obtain a mathematical model for the qPCR experiment that captures both the stochasticity in amplification at each cycle and the uncertainty in fluorescence reads. This project is based on the work of Lalam et al (reference) and uses a binomial branching process to model the amplification. A Hidden Markov Model is used to both capture the transitions between states in this process and relate states to fluorescence observations.

For a qPCR experiment with n cycles, our mathematical model has the following random variables:

  • X0 – the initial number of molecules
  • X1, ..., Xn – the number of molecules present after each amplification cycle
  • F1, ..., Fn – the fluorescence reads observed after each cycle

At each qPCR cycle, only a proportion of the molecules are amplified. In other words, each target mtDNA molecule has a probability r of being amplified, which is the efficiency of the qPCR experiment. This efficiency is assumed to be constant throughout the exponential phase of the amplification curve. A Binomial distribution is used to capture the number of new copies at each cycle, as each molecule undergoes an independent Bernoulli trial with probability r:

Xi+1 = Xi + Binom(Xi , r), where i ≥ 0

The fluorescence intensity emitted by the reporter dye at a given cycle i is proportional to the number of molecules Xi by a constant α. However, the measurements are made in the presence of background fluorescence, which we model as an additive Gaussian noise. The experimental data used for inference is background corrected, so the Gaussian distribution has mean 0 and unknown variance σ 2:

Fi = αXi + N(0, σ 2)

Hidden Markov Model

A Hidden Markov Model (HMM) is a stochastic model with hidden states, where future states only depend on the current one (reference Wikipedia). The specification of a HMM consists of hidden state variables, observations, transition probabilities and observation probabilities.

The binomial branching process is a Markov process, as the number of molecules in each state only depends on the number of molecules in the previous state. We model the process as a HMM with the following specification:

  • X1:n – hidden state variables
  • F1:n – observations
  • transition probabilities given by the binomial with probability r
  • observation probability given by Gaussian noise with variance σ2
The model is parameterized by θ = [X0 , r, σ 2].

The HMM representation influences the choice of inference method that accounts for transitions between states when computing the likelihood of observations for a fixed parameter θ. The particle pseudo-marginal Metropolis Hastings (reference DW) was used to perform inference on the HMM (more in Inference).

The model can be extended to account for other sources of uncertainty, and these states integrated in the inference. For example, the stochasticity in the pipetting procedure can be modelled by adding an extra state *add diagram*

Estimation of the fluorescence coefficient α

The fluorescence coefficient α relates the number of target molecules to fluorescence intensity
Fi = αXi + N(0, σ 2).

Lalam et al treat α as a known constant and do not specify a method to determine its value. However, the quality of parameter inference is highly dependent on the quality of the α estimate as α and X0 are co-dependent when the fluorescence intensity is fixed. In other words, X0 can have any value if α is unrestricted.
